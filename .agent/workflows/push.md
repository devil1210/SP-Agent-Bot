// turbo-all
1. Añadir todos los archivos modificados
`git add .`

2. Crear un commit con los cambios de las 4 verificaciones y gestión de mensajes
`git commit -m "feat: implementar 4 verificaciones y restricciones de gestión de mensajes"`

3. Subir los cambios a la rama principal
`git push origin main`
